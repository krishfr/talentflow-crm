# TalentFlow CRM

Smart Hiring Pipeline Management Platform built for recruitment teams to manage candidates, clients, users, and hiring workflows through a modern full-stack dashboard.

---

## Project Overview

TalentFlow CRM is a full-stack Applicant Tracking System (ATS) and Customer Relationship Management platform designed for recruitment agencies and hiring teams.

The platform helps users manage:

- Candidate pipelines
- Client companies
- User roles and permissions
- Recruitment workflows
- Hiring dashboards
- Secure authentication

This project simulates real SaaS CRM platforms used in the recruitment industry.

---

## Key Features

### Authentication & Security

- JWT based authentication
- User registration and login
- Protected routes
- Persistent sessions
- Secure logout

### Role Based Access Control

- Admin
- Recruiter
- Hiring Manager
- Viewer

Each role receives controlled access to specific modules.

### Candidate Management

- Add candidates
- Track hiring stage
- Search candidate records
- Manage recruitment pipeline

### Client Management

- Add client companies
- Manage open positions
- Maintain company data

### Hiring Pipeline

Track candidate stages:

- Applied
- Interview
- Offer
- Hired

### Dashboard

- Candidate statistics
- Hiring activity overview
- Role-based panels
- Workflow visibility

### UI / UX

- Modern SaaS dashboard
- Responsive design
- Clean layout
- Mobile friendly
